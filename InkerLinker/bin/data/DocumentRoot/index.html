      Node Managers

      Inkerlinker has a numerous managers that handle incoming events from the nodes. This section outlines some of the managers.

      Manager Name Function
      Arduino Manager Handles incoming actions from the arduino node. It also handles the change of pin modes, so when the user changes the output mode of the pin, the manager tells the arduino to do so.
      OSC Manager Handles incoming actions from the osc nodes. By Default the OSC ports are set as localhost and port 12345.
      QLab Manager Handles incoming actions from the qlab nodes. By default the ports are set to localhost port 53000.
